代理標頭的信任級別
使用此設定來確定 Webmin 是否應信任來自代理程式的標頭來識別使用者的 IP 位址和 SSL 憑證：
  • 不，不要相信來自代理商的任何標頭。
  • 是的，信任代理提供的遠端 IP 位址。如果啟用，Webmin 將使用代理提供的標頭來確定瀏覽器的真實 IP 位址，以用於記錄和存取控制目的，例如X-Forwarded-For或X-Real-IP 。僅當您的 Webmin 系統位於代理程式之後，且沒有來自客戶端的直接存取時才應啟用此功能；否則，可能會發送虛假標頭來繞過 IP 存取控制限制。
  • 是的，信任代理提供的遠端 IP 和 SSL 憑證。如果啟用，Webmin 將使用代理提供的標頭來確定使用者的用戶端 SSL 憑證以用於身份驗證，例如X-SSL-Client-DN 。僅當您的 Webmin 系統位於代理程式之後，且沒有來自客戶端的直接存取時才應啟用此功能；否則，可能會發送虛假標頭以其他使用者身分登入。
